# NYTWOLF GAMES

Welcome to the official repository for the **NYTWOLF GAMES** website.

*Note: The initial foundational design was generated using AI Studio, and has since been extensively customized and developed into this complete iteration.*

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
